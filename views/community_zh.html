分享开发经验

分享您已经使用基于 Beego 完成的惊人作品能够很好的帮助他人理解 Beego 的价值。如果您有意愿分享 Beego 的开发故事，我们将乐意为您广而告之。

  • 公司内的技术分享，可以通过微博(#beego, @beego)来通知我们，或者也可以将您的建议或相关会议告知我们，让我们知道您是如何使用 Beego 的。
  • 如果您正在构建基于 Beego 的实际应用，我们将非常乐意地为您传播。
  • 您也可以写一个有关 Beego 的开发教程。

贡献社区代码

一个开源的项目是无法离开开发者社区成员的积极贡献的，到目前为止，已经有超过 15 位贡献者以及 200 多的项目派生。

  • 如果您需要提交问题请通过 Issues List来完成。
  • 您可在 GitHub 上查看我们的发展录像，并给予反馈。
  • 贡献代码，您值得考虑。

我们将 API 文档 也作为一个开源项目来处理，您可以在 Github 的Beego 仓库 中找到相关源码。我们鼓励您参与完善 Beego 的文档，不论改动大小，都欢迎您的补充与提交！
